# Sistema de Chamados T.I.

Esta é uma interface simples e moderna para abrir e gerenciar chamados de T.I.

## Como usar

1. Abra o arquivo `admin/index.html` no navegador.
2. Preencha o formulário para criar um novo chamado.
3. Visualize os chamados enviados e altere o status para `Fechado` ou `Reabrir`.

## Recursos

- Formulário de criação de chamados com nome, setor, prioridade, título e descrição.
- Lista de chamados com status aberto/fechado.
- Persistência local usando `localStorage`.
- Interface responsiva e visual moderno.
